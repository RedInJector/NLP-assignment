
# NLP project

## Prerequisites

### For paper writing

- [Git](https://git-scm.com/install/)
- [VSCode](https://code.visualstudio.com/)
- [Typst compiler](https://typst.app/open-source/#download) 
- [Tinymist Typst](https://marketplace.visualstudio.com/items?itemName=myriad-dreamin.tinymist) extension for VSCode

### For code writing 

- [Miniconda](https://docs.conda.io/projects/conda/en/stable/user-guide/install/index.html)
- Anything that runs jupiter notebooks

## Project overview

- `paper` direcotry contains typst sources for paper generation.
- `src` contains python source code for the project.


## Compiling paper
```sh
cd paper
typst compile main.typ
```
